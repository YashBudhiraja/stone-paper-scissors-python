# Stone Paper Scissors Game

This is a simple Stone Paper Scissors game created using Python. The game allows the user to play against the computer in the terminal, where the computer randomly selects its move and the result is displayed as Win, Lose, or Draw.

## Features

* Play against the computer
* Random computer-generated choices
* Simple game logic using conditions
* Beginner-friendly Python project

## Concepts Used

* Dictionaries
* Conditional statements
* Random module
* User input handling
